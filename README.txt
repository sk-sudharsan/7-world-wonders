Simple Webpage
This is a simple project demonstrating the creation of a  webpage using just CSS and HTML.
This project simply demonstrates the functionality of hyperlinks, image tags, and list tags.




<a>---anchor tag-> create hyperlink to webpages
<img>---image tag-> used to embed an image in an HTML page
<ul>--unordered list-> used to create unordered lists
<div>--division tag-> defines a devision or section in an HTML
